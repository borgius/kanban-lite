# Kanban Markdown

A kanban board for your codebase. Features stored as markdown.

## Features

- Kanban board with drag-and-drop
- Feature cards with priority, labels, assignee, and due dates
- Features stored as markdown files in `.devtool/features/`
- Integrates with VSCode/Cursor theme

## Usage

1. Open command palette (`Cmd+Shift+P`)
2. Run "Open Kanban Board"
3. Start shipping

## Development

```bash
pnpm install
pnpm build
```

Press F5 to launch Extension Development Host.
